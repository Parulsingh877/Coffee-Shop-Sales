# ☕ Coffee Shop Sales Dashboard

## 📌 Overview

This project analyzes coffee shop transaction data using **Microsoft Excel**. The dashboard helps track sales performance, transaction trends, product performance, and store-level results.

## 🛠️ Tools Used

* Microsoft Excel
* PivotTables
* PivotCharts
* Slicers
* Excel formulas
* KPI cards
* Data cleaning and formatting

## 📊 Key KPIs

* **Total Sales:** $698,812.33
* **Total Transactions:** 149,116
* **Average Order Value:** $4.69
* **Average Items per Transaction:** 1.44

## 📈 Dashboard Analysis

The dashboard includes:

* Quantity sold by hour
* Sales distribution by product category
* Transaction distribution by product size
* Transactions and sales by store location
* Top 5 products by sales
* Transactions and sales by day of week

## 🔍 Key Insights

* Barista Espresso is one of the top-selling products.
* Sales remain fairly consistent across store locations.
* Weekday sales are slightly higher than weekend sales.
* Peak-hour analysis can help improve staff scheduling.
* Product bundles can help increase average items per transaction.
